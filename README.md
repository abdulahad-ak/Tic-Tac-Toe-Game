# Tic-Toc-Toe-Game

A beginner-level Tic Tac Toe game developed with HTML, CSS, and JavaScript, emphasizing DOM manipulation, user interaction, and fundamental game logic.

## Features
- Two-player gameplay
- Win and draw detection
- Restart game option
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project was built to practice DOM manipulation and JavaScript fundamentals.
